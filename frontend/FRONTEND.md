# TrafiVista Frontend Documentation

Welcome to the **TrafiVista** frontend documentation. This document provides an overview of the architecture, tech stack, and key components of the web application.

## 🚀 Tech Stack

- **Core**: [React 18](https://reactjs.org/) with [TypeScript](https://www.typescriptlang.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) + [Shadcn UI](https://ui.shadcn.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **State Management**: React Hooks + [TanStack Query](https://tanstack.com/query/latest) (for future API integration)
- **Routing**: [React Router v6](https://reactrouter.com/)

## 🎨 Design System

TrafiVista uses a **Deep Dark / Glassmorphism** aesthetic:
- **Primary Color**: `#8B5CF6` (Vivid Purple)
- **Background**: Dark charcoal/black with subtle grid patterns.
- **Glass Effects**: High levels of backdrop-blur and semi-transparent borders.
- **Responsive**: Fully responsive design (RWD) using Tailwind's breakpoint system.

## 🛠 Key Features (Frontend)

### 1. Hierarchical Location Selection
Implemented in `UserUpload.tsx`, allowing users to select locations in India via a multi-step dropdown (State → District → City → Spot).

### 2. Prediction Interface
- **Date/Time Picker**: Professional calendar integration for selecting future forecast targets.
- **File Validation**: Client-side name/keyword check to ensure uploaded files are traffic-related before allowing predictions.

### 3. Admin Dashboard (Implementing)
- **Auto-Training**: Simulation of model training triggered by dataset uploads.
- **Model Versioning**: History system to "Undo" or restore previous model states.

## 📖 Component Guidelines

- **Typography**: Uses `Inter` for body text and a custom `font-display` for headings.
- **Icons**: Always use `Lucide` icons for consistency.
- **Toasts**: Use `sonner` for all system notifications and feedback.
- **Transitions**: Every page transition and modal uses `Framer Motion` for a premium feel.
